What is the correct way to declare a variable in Python?

a. var x = 5
b. x = 5
c. int x = 5
d. x := 5
Correct: b

Which of the following is a valid Python comment?

a. /* This is a comment */
b. // This is a comment
c. # This is a comment
d. -- This is a comment
Correct: c

How do you check the length of a list in Python?

a. len(list)

Which keyword is used for creating a function in Python?

a. function

What is the purpose of the if statement in Python?
